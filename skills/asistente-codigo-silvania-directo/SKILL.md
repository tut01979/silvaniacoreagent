## Asistente de Código Silvania (Acceso Directo - ¡PELIGRO!)

**Descripción:** Esta habilidad permite a Silvania CoreAgent acceder a tu ordenador para analizar y modificar código fuente, solicitando autorización en cada paso.

**¡ADVERTENCIA EXTREMA!:** Esta habilidad implica RIESGOS DE SEGURIDAD GRAVÍSIMOS. Úsala BAJO TU PROPIO RIESGO y solo si comprendes las implicaciones. Silvania CoreAgent NO se hace responsable de los daños que pueda causar esta habilidad.

**Instrucciones:**

1.  Proporciona la siguiente información a Silvania CoreAgent:
    *   Tu nombre de usuario en el sistema.
    *   Tu contraseña (¡NUNCA proporciones tu contraseña a menos que estés absolutamente seguro de lo que estás haciendo!).

2.  Silvania CoreAgent intentará acceder a tu ordenador utilizando las credenciales proporcionadas.

3.  Silvania CoreAgent solicitará autorización para cada acción que realice (ej: leer un archivo, modificar un archivo, ejecutar un comando).

**Herramientas disponibles:**

*   Acceso a la terminal del sistema.
*   Modelo de lenguaje Gemini Pro (para análisis de código).
*   Intérprete de código (para ejecutar código en un entorno seguro).

**Ejemplo de uso:**

Usuario: "Conéctate a mi ordenador (usuario: jesus)"

Silvania CoreAgent: "¿Cuál es tu contraseña?"

Usuario: "(Proporciona la contraseña)"

Silvania CoreAgent: "¿Puedo ejecutar el comando `ls /home/jesus/proyectos` en tu ordenador? (Sí/No)"

Usuario: "Sí"

Silvania CoreAgent: "(Muestra la lista de archivos) ¿Puedo abrir el archivo `main.py` para analizarlo? (Sí/No)"

Usuario: "Sí"

Silvania CoreAgent: "(Muestra el contenido del archivo) He detectado un posible error en la línea 10. ¿Puedo modificar el archivo para corregirlo? (Sí/No)"

Usuario: "No"

**Consideraciones de seguridad:**

*   ¡NO proporciones tu contraseña a menos que estés absolutamente seguro de lo que estás haciendo!
*   Supervisa cuidadosamente las acciones de Silvania CoreAgent.
*   Cambia tu contraseña inmediatamente después de utilizar esta habilidad.
*   Desactiva la habilidad cuando no la estés utilizando.
*   Crea copias de seguridad de tus archivos importantes antes de utilizar esta habilidad.
